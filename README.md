# Launch Big Brands - GitHub Pages Repository

This repository hosts the static website for Launch Big Brands at `lbbgo.com`.

## Structure

- `index.html` - Homepage with full landing page
- `video/` - Individual video pages with clean URLs
- `assets/` - Shared CSS, JS, fonts, and images
- `screenshots/` - Website screenshots for video players

## Clean URLs

Video pages are structured as:
- `lbbgo.com/video/nov107/` (instead of `nov107.html`)
- `lbbgo.com/video/nov108/` (and so on...)

## Deployment

This site is automatically deployed via GitHub Pages when changes are pushed to the main branch.

## Domain Setup

The custom domain `lbbgo.com` is configured via the `CNAME` file and GitHub Pages settings.
